### Shortcuts

Becoming a super user is a fairly straight forward process: just remember and use shortcuts

| Shortcut | Description |
| :--- | :--- |
| ⌘ ↵ | Send Request |
| ⌘ S | Save Request |
| ⌘ L | Request URL |
| ⌘ \ | Toggle Sidebar |
| ⌘ E | Rename Item |
| ⌘ D | Duplicate Item |
| ⌘ + | Zoom In |
| ⌘ - | Zoom Out |
